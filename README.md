# Sistema de Gestão de Fluxo Clínico

## ⚠️ Status do Projeto: Não Iniciado

Este repositório contém a base estrutural e os artefatos iniciais para o **Sistema de Gestão de Fluxo Clínico**, um projeto concebido para a disciplina de *Estrutura de Dados I*.

**Atenção:** O desenvolvimento ativo (escrita de código da aplicação) **ainda não começou**. Os arquivos aqui presentes são apenas de planejamento (como o documento `.docx` base) e o esqueleto do projeto para configurações iniciais de ambiente.

### 🎯 Objetivo Futuro
Quando iniciado, o projeto implementará conceitos acadêmicos na linguagem de programação para gerenciar a fila de pacientes usando:
- **Listas Encadeadas** (para cadastros gerais)
- **Filas / FIFO** (para ordem de triagem e chegada)
- **Pilhas / LIFO** (para controle do histórico da consulta médica)
- Adicionalmente, métricas de complexidade computacional para algoritmos de **Ordenação e Busca**.

### 🛠️ Tecnologias Planejadas
- Interface interativa
- Algoritmos clássicos de Estrutura de Dados
